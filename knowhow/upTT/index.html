Виджет кнопки вверх как ВКонтакте

Подключение виджета на Ваш сайт (секция head)

Кнопка выполнена в том же стиле, что и аналогичная кнопка в популярной социальной сети ВКонтакте.


<!--up button--><script type="text/javascript" src="/knowhow/jslibs/jquery-1.5.min.js"></script>
<script type="text/javascript" src="/knowhow/jswidgets/upTT.js"></script><!--end../ up button-->

В данном коде можно изменить некоторые параметры под себя 


$(document).ready(function() {
	
	$('body').append('<div class="button-up" style="display:none; opacity:0.7; width:85px; height:100%; position:fixed; left:0px; top:0px; cursor:pointer; text-align:center; line-height:100px; color:rgb(172, 188, 206); font-weight:100;"><font size="2">&#9650; Наверх</font></div>');
		
	$ (window).scroll (function () {
		if ($ (this).scrollTop () > 700) {
			$ ('.button-up').fadeIn();
		} else {
			$ ('.button-up').fadeOut();
		} 
	});
	
	$('.button-up').click(function(){
		$('body,html').animate({
            scrollTop: 0
        }, 1400);
        return false;
	});
	
	$('.button-up').hover(function() {
			$(this).animate({
				'opacity':'1',
			}).css({'background-color':'#e7ebf0','color':'#6a86a4'});
		}, function(){
			$(this).animate({
				'opacity':'0.7'
			}).css({'background':'none','color':'#d3dbe4'});;
	});
		
});
  • Строка 2 отвечает за вывод самой области для прокрутки страницы. В ней прописаны основные стили для отображения кнопки. в принципе, данные параметры должны подойти практически каждому.
    Но может потребоваться и отредактировать их под себя;
  • В строке 4 в скобках имеется число 700, которое отвечает за момент появления кнопки. То есть, кнопка будет появляться только после прокрутки 700 пикселей вниз.
    Рекомендую тут подобрать такое значение, чтобы прокрутка была возможной только тогда, когда с поля зрения пропадает основное меню сайта или какие-то другие важные элементы в навигации;
  • В строке 13 значение 1400 отвечает за скорость прокрутки. Чем меньше значение, тем быстрее возвращение вверх страницы.
