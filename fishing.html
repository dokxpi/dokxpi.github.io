Как: скрыть "носители" через реестр Windows NT

Скрытие отдельных физических, съемные или оптических приводов в Моем компьютере можно в Windows с помощью "NoDrives" запись реестра. Эта запись скрывает указанные диски с моего компьютера, Windows Explorer, и в меню Отправить.

Шаги для выполнения этой операции приведены ниже:

  1. Запустите редактор реестра (Win + R > regedit > ОК)
  2. Откройте следующий раздел реестра:
    1. для текущего пользователя 
      HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer
    2. для всех пользователей HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer
  3. Создайте параметр NoDrives типа DWORD (DWORD (32 бита))
  4. Щелкните правой кнопкой мыши по параметру и выберите пункт Изменить...
    1. Установите переключатель на значение Десятичная
    2. В текстовое поле введите значение/число, полученное из калькулятора: 67108863

Как: запретить доступ к "носителям" через реестр Windows NT

  1. Запустите редактор реестра (Win + R > regedit > ОК)
  2. Откройте следующий раздел реестра (Правка > Найти (CTRL+F)):
    1. для текущего пользователя HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer
    2. для всех пользователей HKEY_LOCAL_MACHINE\Software\Microsoft\Windows\CurrentVersion\Policies\Explorer
  3. Создайте параметр NoViewOnDrive типа DWORD (Параметр DWORD (32 бита))
  4. Щелкните правой кнопкой мыши по параметру и выберите пункт Изменить...
    1. Установите переключатель на значение Десятичная
    2. В текстовое поле введите значение/число, полученное из калькулятора
